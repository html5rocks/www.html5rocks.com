{% trans "Try This!" %}

{% blocktrans %}CSS3 Transformations allow you to apply rotations, scales, skews and translations to any DOM element with simple styles.{% endblocktrans %}

pre {
transform: rotate(-1deg) skewX(5deg);
}

{% blocktrans %}Transitions allow you to specify how the style on a DOM element will animate between transformed states. For example, you can animate the background color when the user hovers over an element. Hover over this code!{% endblocktrans %}

pre {
  transition: all 1s ease-inout;
}

pre:hover {
  background-color: #F2F5FE;
  border-color: black;
}

{% blocktrans %}Applying 3D transforms is simple too. Just specify the transformation in 3d co-ordinates and you are ready. Hover over this code!{% endblocktrans %}

div {
  transform: rotate3d(1,1,0, 45deg);
}
